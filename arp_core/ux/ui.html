Help, quick links, and more info

  • You can view important keybinds and text commands via /keybinds.
  • You must be connected to our TeamSpeak server via ts.australianroleplay.com and have the TokoVoip plugin installed. See the Support Centre on the Community Portal for more information.
  • You can change your character selection via the Service Portal (it will change when you relog).
  • Bugs can be reported via the Bug Tracker in the Community Portal.
